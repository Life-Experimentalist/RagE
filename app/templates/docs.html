About RAG

Retrieval-Augmented Generation (RAG) is a technique that enhances large language models by retrieving relevant information from a knowledge base before generating responses.

This approach improves accuracy and ensures up-to-date information.

API Documentation

Query the RAG System

POST /api/rag/query
{
    "query": "Your question here",
    "max_results": 3
}

Add Documents

POST /api/rag/documents
{
    "documents": [
        {
            "content": "Document text content",
            "source": "Source information"
        }
    ]
}
